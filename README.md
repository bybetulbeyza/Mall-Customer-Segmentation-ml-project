# Mall Customer Segmentation using K-Means

This project demonstrates the application of **unsupervised machine learning** techniques to perform customer segmentation.

## Objective
To group customers into meaningful segments based on purchasing behavior using **K-Means clustering**, enabling data-driven marketing insights.

## Dataset
- Source: Kaggle (Mall Customers Dataset)

## Approach
- Exploratory data analysis
- Feature selection
- Elbow Method to determine optimal number of clusters
- K-Means model training
- Cluster visualization

## Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Purpose
This project was developed for learning and portfolio purposes to gain hands-on experience with clustering algorithms and data visualization.
